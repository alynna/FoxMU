*** FoxMU 0.9 Help file ***

If you cannot see the whole page in the preview, click the link to open the Help in a new window.

Icons:


Log file		 This displays the log of all text received so far. Its in HTML format to preserve image links and colors. To make a text log out of it, save the text to a file, open it in your browser, then either cut and paste or save it as text.

Settings		 This menu contains various settings you can change. Most are saved in your cookies. You can also access some of the help files and sites in this menu.

 Status		 This menu shows your connected/disconnected status (Red if disconnected, Green if connected). Additionally, a menu for connecting, disconnecting and reconnecting resides in here.

Things on the settings menu:

Help menu This menu leads to pages for help, including this help. The command reference will have clickable commands that work in the game, and the help page for the game should also be here.
Auto Login Auto-login allows you to store credentials for logging in, in your cookie, so that it will automatically log in when you connect. To log into another character, you can clear the cookie by pressing 'cancel', or store different credentials in there. If you want to log into another character from the MUCK, try the 'relog' command.
Command Echo If you want to have your commands echoed to the game, you can turn Command Echo on (or off).
Favorite Font You can change the font you want to use here. This setting is saved. To resize the fonts, you can use your zoom command in your browser (typically, hold shift while using your scroll wheel).
Clear Screen If what is on your screen offends you, you may use this function to clear the screen. Or if alot of images are slowing down your experience, you can also partially clear your buffer.

Additional Info:

  • For your convenience, URLs are automatically converted to clickable links. They will open in a new window if clicked.
  • If you hover over an image link, a larger preview will be displayed on the right top side of your browser window.
  • You can inline an image into your input. Simply link the image as per normal, except replace 'http' with 'img' like 'http://image.com/pic.png' to 'img://image.com/pic.png'.
